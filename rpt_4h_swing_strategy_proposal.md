# 4시간봉 스윙 전략 최적화 파라미터 제안 리포트 (추후 적용용)

## 1. 개요
이 리포트는 기존 15분봉(15m) 기반 단타 전략의 잦은 매매로 인한 수수료 누수 문제를 해결하기 위해, 100회 백테스트 최적화를 통해 도출된 **4시간봉(4h) 스윙 전략** 파라미터 제안을 정리한 문서입니다.
현재 설정(15m)을 며칠 더 유지하며 관찰한 뒤, 추후 스윙 전략으로 전환할 때 참고하기 위해 작성되었습니다.

## 2. 최적 파라미터 비교 (권장 설정값)

| 파라미터 (설정 변수) | 기존 설정 (현재) | **4h 최적 권장값** | 변경 사유 및 기대 효과 |
| :--- | :---: | :---: | :--- |
| **차트 타임프레임** | 15m (15분봉) | **4h (4시간봉)** | 수수료 방어 및 큰 추세 수익 창출 (가장 핵심적인 변경) |
| **손절 비율 (SL)** | 1.0% | **3.62%** | 4시간봉의 큰 변동성에 의해 휩소(가짜 하락)로 손절되는 것을 방지 |
| **익절 비율 (TP)** | 1.5% | **4.95%** | 손절폭(3.6%)을 늘린 만큼 보상 비율(손익비)을 맞추어 한 번에 크게 익절 |
| **레버리지** | 10x | **10x** | 시뮬레이션상 20x가 수익률이 높으나, 자본금 보존 및 안정성을 위해 기존 10x 유지 권장 |
| **1회 진입 증거금** | 10.0 USDT | **16.1 USDT** | 자산 배분 측면에서 16 USDT 부근 진입이 복리 수익 극대화에 유리 |
| **최대 동시 포지션** | 4개 | **4개** | 유지 (전체 자본금 리스크 관리) |

## 3. 전체 봇 진입 횟수와 제한 로직 (안전장치)
* **신호 발생:** 80여 개 전 종목을 4시간봉으로 감시하면 전체 시장에서 하루 평균 40~120번의 조건 부합 '신호'가 발생할 수 있습니다.
* **진입 제한 (MAX_POSITIONS):** 하지만 `core/config.py`의 `MAX_POSITIONS = 4` 설정으로 인해 4개의 포지션 슬롯이 가득 차면 추가 신호는 모두 진입을 무시합니다.
* **순환 횟수:** 4시간봉 스윙 전략 특성상 한 포지션당 반나절~하루 이상 유지됩니다. 즉, 4개의 슬롯이 하루에 1~2바퀴 정도만 순환합니다.
* **최종 매매 횟수:** 따라서 80개 종목 전체를 대상으로 하더라도, **실제 봇의 총 진입 횟수는 하루 4~10회 내외**로 안전하게 억제됩니다. 잦은 매매로 인한 수수료 폭발 걱정은 없습니다.

## 4. 시뮬레이션 예상 결과 (100회 평균)
* **일 평균 수익률:** 3.09%
* **최대 낙폭 (MDD):** 4.15% (매우 안정적인 수치)
* **승률:** 66.7%

## 5. 적용 방법
추후 이 전략을 적용할 때는 `core/config.py` 파일을 열어 `TradingConfig` 클래스 내부의 파라미터 값들을 위의 4h 권장값으로 수정한 후 봇을 재시작하면 됩니다.
